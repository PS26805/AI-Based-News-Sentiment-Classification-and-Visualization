# 📰 AI-Based News Sentiment Classification and Visualization

## 📌 Project Overview
This project is an AI-powered News Sentiment Analysis Dashboard developed using Python and Streamlit. It analyzes news headlines and classifies them into Positive, Neutral, or Negative sentiments while providing interactive visualizations for better insights.

## 🚀 Features

- 📊 Interactive Dashboard
- 😊 Positive, Neutral & Negative Sentiment Analysis
- 📈 Sentiment Distribution Chart
- 🥧 Sentiment Breakdown Pie Chart
- 📂 Category-wise Analysis
- 🔍 Search Headlines
- ☁️ Word Cloud Visualization
- 📥 Download Filtered Dataset
- 📋 Dataset Preview
- 📊 Project Statistics

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- WordCloud

## 📂 Dataset

News Sentiment Analysis Dataset (CSV)
